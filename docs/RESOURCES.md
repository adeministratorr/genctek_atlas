# Resources & Libraries

## 1. Libraries and Packages

- **[React](https://react.dev/):** UI framework.
- **[Vite](https://vitejs.dev/):** Next Generation Frontend Tooling.
- **[Firebase SDK](https://firebase.google.com/docs/web/setup):** For Auth, Firestore, and Storage.
- **[Lucide React](https://lucide.dev/):** Lightweight, customizable SVG icons.
- **[Leaflet](https://leafletjs.com/) / [React-Leaflet](https://react-leaflet.js.org/):** (Optional) Used for advanced geographic mapping features.

## 2. Tools & Platforms

- **Version Control:** Git & GitHub.
- **AI Coding Platforms:** Google AI Studio or preferably Google Antigravity.
- **Recommended AI Model:** Gemini 3.5 Flash (Optimized for speed and long-context Vibe Coding).
- **Deployment:** Firebase Hosting.

## 3. External Assets & Data Sources

- **SVG Map:** Turkey SVG paths extracted and customized from open-source maps (e.g., Wikimedia Commons Blank Map of Turkey).
- **GeoJSON Data:** The polygon data for Turkey's 81 provinces used in Leaflet was sourced from open-source GitHub repositories (e.g., [alpers/Turkey-Maps-GeoJSON](https://github.com/alpers/Turkey-Maps-GeoJSON)).
- **Fonts:** [Google Fonts](https://fonts.google.com/) (Inter or Roboto).

# Kaynaklar & Kütüphaneler (Resources)

## 1. Kütüphaneler ve Paketler

- **[React](https://react.dev/):** Kullanıcı Arayüzü (UI) iskeleti.
- **[Vite](https://vitejs.dev/):** Yeni nesil, süper hızlı frontend derleme aracı.
- **[Firebase SDK](https://firebase.google.com/docs/web/setup):** Auth, Firestore ve Storage işlemleri için.
- **[Lucide React](https://lucide.dev/):** Hafif ve özelleştirilebilir SVG ikon seti.
- **[Leaflet](https://leafletjs.com/) / [React-Leaflet](https://react-leaflet.js.org/):** (Opsiyonel) Gelişmiş coğrafi haritalama özellikleri için.

## 2. Araçlar ve Platformlar

- **Sürüm Kontrolü:** Git & GitHub.
- **Yapay Zeka Kodlama Platformları:** Google AI Studio veya tercihen Google Antigravity.
- **Önerilen Yapay Zeka Modeli:** Gemini 3.5 Flash (Hızlı tepki süresi ve Vibe Coding için optimize edilmiştir).
- **Canlıya Alma (Deployment):** Firebase Hosting.

## 3. Dış Kaynaklar ve Veri Setleri

- **SVG Harita:** Açık kaynaklı dilsiz haritalardan (örn. Wikimedia Commons dilsiz Türkiye haritası) alınıp React bileşenine dönüştürülen Türkiye SVG çizimleri.
- **GeoJSON Verisi:** Leaflet haritasında kullanılan 81 ilin poligon koordinat verileri, açık kaynaklı GitHub depolarından (örn. [alpers/Turkey-Maps-GeoJSON](https://github.com/alpers/Turkey-Maps-GeoJSON)) alınarak projeye entegre edilmiştir.
- **Yazı Tipleri:** [Google Fonts](https://fonts.google.com/) (Önerilen: Inter veya Roboto).
